## :wrench: Installation

__You must install app dependencies using the following command on your Terminal:__

```bash
# yarn
$ yarn install

# npm
$ npm install
```

__Execute the following command to start app:__

```bash
# yarn
$ yarn start

# npm
$ npm start
```
